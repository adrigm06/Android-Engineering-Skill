# Regression Evaluation Rubric

Score each response from 0 to 2 on every criterion.

- `0` = missing or incorrect
- `1` = partial
- `2` = complete and correct

## Criteria (max score: 16)

1. **Routing accuracy**: identifies correct lead domain and relevant supporting domains.
2. **Constraint compliance**: preserves non-negotiables and boundary rules.
3. **Authority handling**: applies conflict precedence when tradeoffs exist.
4. **Evidence orientation**: uses metrics, gates, or a measurement-first path.
5. **Confidence protocol**: provides confidence band and assumptions when needed.
6. **Actionability**: gives ordered, implementable steps.
7. **Integrated synthesis**: returns one coherent plan, not disconnected checklists.
8. **Uncertainty handling**: asks for or states minimum missing data when ambiguous.

## Outcome bands

- `14-16`: Production-ready behavior
- `11-13`: Usable, improve weak areas
- `0-10`: Needs refactor
