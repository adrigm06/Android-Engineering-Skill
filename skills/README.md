# Skills Catalog

This directory contains domain-specific skills.

## Core skills (v1)

- `android-architecture`: architecture strategy, modularization, boundaries, ADRs
- `android-compose`: UI state modeling, recomposition and performance guidance
- `android-gradle-build`: build logic, dependency management, and build speed
- `android-testing`: test strategy, scope, reliability, and benchmark-oriented testing
- `android-performance`: startup/jank/memory/ANR/battery optimization workflows
- `android-security`: mobile threat-aware engineering recommendations

## Additional skills (v1)

- `android-debugging`

Use only the relevant skill(s) for the task to keep context focused.
