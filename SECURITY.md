# Security Policy

## Supported scope

This repository primarily contains skill definitions and documentation.
Security reports are still welcome, especially for:

- Unsafe recommendations in security-related skills
- Guidance that could cause credential leakage
- Release/build advice that weakens Android app security posture

## Reporting a vulnerability

Please report privately to:
maintainers@android-engineering-skill-package.org

Include:

- Affected file(s)
- Prompt scenario
- Observed risky recommendation
- Potential impact
- Suggested mitigation (if available)

## Response targets

- Initial acknowledgment: within 5 business days
- Triage update: within 10 business days

## Disclosure

Please avoid public disclosure until maintainers confirm triage and remediation status.
