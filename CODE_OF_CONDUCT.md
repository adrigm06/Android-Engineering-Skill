# Code of Conduct

## Our commitment

We are committed to a respectful, inclusive, and constructive community.

## Expected behavior

- Be respectful in technical disagreements
- Give actionable feedback
- Assume good intent
- Prefer evidence over opinion
- Keep discussions focused on engineering quality

## Unacceptable behavior

- Harassment, discrimination, or hate speech
- Personal attacks or insults
- Deliberate misinformation
- Repeatedly derailing technical discussions

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

For concerns, contact: maintainers@android-engineering-skill-package.org

## Scope

This code of conduct applies in repository issues, pull requests, and project discussions.
